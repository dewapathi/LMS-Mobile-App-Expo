Step 2: Reinstall Expo Go APK in Emulator
Run this to uninstall the possibly broken version:

adb uninstall host.exp.exponent

Then reinstall your APK:
adb install ~/Downloads/Exponent-2.31.2.apk
Then, manually open Expo Go again.

//get ip address - mac
ipconfig getifaddr en0

//app Run
npx expo start -c
npx expo start
